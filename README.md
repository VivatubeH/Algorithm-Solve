# 알고리즘 문제 풀이

프로그래머스 기준 코딩테스트 합격권 기초 체력을 만드는 것을 목표로 하는 학습 레포입니다.

기존 백준 풀이 자산은 그대로 활용하고, 앞으로는 프로그래머스 중심으로 이론 학습과 문제 풀이를 병행합니다.

## 목표

- 자료구조/알고리즘 핵심 개념을 설명할 수 있다.
- 프로그래머스 Level 1~2 핵심 유형을 스스로 해결할 수 있다.
- 제한 시간 안에 문제를 읽고, 접근법을 정하고, 구현까지 마무리할 수 있다.
- 풀이 후 회고를 남기며 실수를 줄여 나간다.

## 학습 원칙

1. 이론을 짧게 학습하고 바로 문제에 적용한다.
2. 문제를 풀 때는 정답 암기보다 "왜 이 풀이가 맞는지"를 설명할 수 있어야 한다.
3. 한 문제를 풀고 끝내지 않고, 실수와 개선 포인트를 문서화한다.
4. 어려운 문제보다 자주 나오는 유형을 먼저 확실히 잡는다.
