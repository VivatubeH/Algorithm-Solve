# 📂 [Algo] 백준 2417 - 정수 제곱근

> 날짜: 2026-01-27 | 난이도: Silver 4

### 1. 핵심 아이디어 💡
* **알고리즘**: `판정함수 : q^2 >= N`, 매개변수 탐색
* **핵심 발상**:
* - **선형 탐색** : O(root N) : 시간 초과 ( 최악의 경우 2^31이 넘어감 )
* - **매개변수 탐색** : O(log n) : 시간 내에 가능

### 2. 코드 리뷰 & 로직 ⭐
* **주의점**: 값의 범위가 크기 때문에 안전하게 long 타입의 변수를 사용한다.
* **핵심 수식**: `mid * mid >= n`을 `mid >= n / mid`로 전환하면서 동시에 나머지 처리하기.

### 3. 트러블슈팅 & 배운 점 🚩
- 제곱이 들어가기 때문에, 범위를 벗어날 위험을 고려해야 함.
- 판정함수에서 0이 들어올 경우에 대한 예외 처리 필요. ( 0으로 나누면 안됨 )
- double의 정밀도를 15자리로 가정했을 때, 무턱대고 double로 변환은 위험함.
- `1 << 32` 비트 연산을 통해 연산을 최적화한다.
