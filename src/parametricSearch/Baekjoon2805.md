# 📂 [Algo] 백준 2805 - 나무 자르기

> 날짜: 2026-01-27 | 난이도: Silver 2

### 1. 핵심 아이디어 💡
* **알고리즘**: 매개변수 탐색
* **핵심 발상**: 
* - 매개변수 탐색 : O(N * logM) - 시간 복잡도내에 가능
* - M미터 이상의 나무를 가져가기 위해 ( 판정 함수 )
* - 설정할 수 있는 절단기 높이의 최댓값 ( 이분 탐색 범위 조정 )

### 2. 코드 리뷰 & 로직 ⭐
* **주의점**: 나무 높이의 합을 long으로 둬야 함. 
* **핵심 수식**: 최적화라면, 나무를 자르다가 M보다 크거나 같아지면 바로 종료해도 됨. 

### 3. 트러블슈팅 & 배운 점 🚩
- 범위를 고려해서 자료형을 선택했다. upper가 10억이하이므로, mid가 int 초과 불가.
- 판정 중간에 만족하면 탈출할 수 있게 최적화 했다.
