- 유형 : 문자열
- 소금 폭탄

##### 풀이
1. 로봇팔이 소금을 투하할 때까지 시간은, 1초보다 크거나 같고, 24시간보다 작거나 같음.
- 즉, 00:00:01 ~ 24:00:00 사이로 숫자가 나와야 함.
2. 첫 번째 줄은 현재 시간, 두 번째 줄은 투하 시간 ( 종료 시간 ) 으로 생각 시 소요 시간을 구하는 문제
3. 예외 케이스
- 현재 시간 >= 투하 시간 : 투하 시간 - 현재 시간 + 24시간
- 현재 시간 < 투하 시간 : 투하 시간 - 현재 시간

##### 포인트
1. 시분초 계산이 복잡할 때, 전부 초로 변환 후 계산하면 편리하다.
- 시간 : 3600초, 분 : 60초
2. 문자열 형태로 :을 구분해서 주어질 때, 시분초를 얻어오는 방법
- split 함수를 통해 ":"를 기준으로 시, 분, 초를 따로 얻어온다.
3. 출력 시 0이 앞에 붙게 하려면 printf의 %02d를 사용한다. ( %02d : 두 자리 확보, 빈곳은 0으로 채움 )
