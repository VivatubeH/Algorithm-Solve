- 유형 : 문자열
- 단어 공부

##### 풀이
1. 가장 많이 사용된 알파벳 알기 -> 빈도 체크
2. 단, 대소문자 구분 x이므로 체크 시 처리 필요.
3. 대문자나 소문자로 변환 후 -> 빈도 체크
4. 최장 빈도 알파벳 출력하기 ( 여러 개일 경우, ? 출력 )

##### 포인트
1. toLowerCase, toUpperCase 사용이 좋을 것 같음.
2. 가장 많이 사용된 알파벳 여러 개 체크를 어떻게 할지?
- 빈도가 클 때만 갱신하면 : 같은 경우가 2개 이상 시 체크가 불가함.
- 빈도가 크거나 같을 때 갱신하면 : 같을 시에 ?로 변환하면, 이후 클 시 해당 알파벳으로 변환됨.
3. 출력 조건 = 가장 많이 사용된 알파벳을 '대문자'로에 주의
- 즉, 처음부터 대문자로 변환 (toUpperCase) 후 체크 및 출력이 좋음.
4. 조건문이 중첩된 경우 -> 값 변경이 미칠 영향을 고려해서 코드 짜기.
- 예를 들어, if문 내에서 값을 변경한 후에 변경된 값을 기준으로 조건 검사는 안됨.
- 조건 검사를 먼저하고, 이후 변경이 이루어지도록 습관 들이기.
