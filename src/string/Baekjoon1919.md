- 유형 : 문자열
- 애너그램 찾기 
- 애너그램 : 두 단어가 철자 순서를 바꾸어서 같아질 수 있는 관계

##### 풀이
1. 순서를 바꾸는 횟수는 제한이 없기 때문에, 모든 알파벳의 등장 빈도가 1:1 대응되면 애너그램
2. 각 단어별로 알파벳 등장 빈도를 체크해서 배열에 저장해두고,
3. 배열에서 알파벳 등장 빈도가 다른 경우, 더 작은 등장 빈도와 같게 맞춰준다.
- 예를 들어, 등장 빈도가 3과 2라면 1개를 제거한 게 제거해야 하는 문자수가 최소이다.
- 이러한 제거된 개수를 변수에 누적 후, 마지막에 출력하면 된다.

##### 포인트
1. 애너그램이 무엇인지 이해하기 ( 핵심은, 알파벳별 등장 빈도가 같다면 애너그램이 가능하다. )
2. 제거하기 위한 최소의 문자 개수 = 더 작은쪽에 맞게 큰 빈도 - 짝은 빈도를 계속 누적하기.
3. if문으로 조건문 분기를 할 수도 있지만, 절댓값을 이용하면 조건문 없이 덧셈으로만 처리 가능.
