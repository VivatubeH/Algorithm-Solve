- 유형 : 문자열
- 대소문자 변환 문제
- 대문자와 소문자의 아스키 코드값 차이를 이용해서 변환

##### 풀이
1. Scanner 클래스 사용 시, util 클래스 import 하기.
2. nextLine() 메서드로 한 줄의 문자열 입력받기.
3. 'a' - 'A'를 통해서 두 문자의 아스키 코드값 차이 미리 구해두기.
4. 입력된 문자열을 하나씩 순회하면서
- 소문자이면, 아스키 코드값 차이만큼 빼서 출력하기
- 대문자이면, 아스키 코드값 차이만큼 더해서 출력하기

##### 셀프 피드백
1. a = 97, A = 65로 기본적인 알파벳 아스키 코드값 2개는 외워두면 편함.
2. int 보다 작은 타입끼리 사칙연산은 int로 자동 형변환되므로 출력시 char 변환 필수. 
3. 알파벳만 주어진다면 if ~ else로도 가능 ( 소문자가 아니면 무조건 대문자 )
4. 해당 조건 없을 시 , if ~ else if 사용이 나을 듯. ( 대문자인지, 소문자인지 체크필수 )
6. if문 여러 개일 시, 오류 발생 가능성 존재. ( 이 문제에서는 해당 X )
