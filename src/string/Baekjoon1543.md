- 유형 : 문자열
- 문서 검색 문제

##### 풀이
1. 영어로만 이루어진 문서
2. 단어를 중복되게 세서는 안됨. 
3. 단어가 최대 몇 번 중복되게 등장하는 지 빈도를 세야함.

##### 포인트
1. 해당 단어가 포함되었는지 세려면? -> contains, indexOf 활용 가능
2. 단, 이전에 등장하면 중복세기 불가 -> indexOf로 시작 위치 + 단어 길이가 다음 시작위치임.
- 예를 들어, abcabc에서 abc를 찾는다면 abc의 시작위치 0 + 단어 길이 3인 3이 다음 시작위치임.
3. 이 과정을 indexOf의 결과값이 -1이 아닌 동안 수행하면됨. ( indexOf는 존재하지 않을 시 -1를 반환 )
4. Scanner.next()는 공백을 기준으로 끊기 때문에 이 문제에서는 nextLine()을 써야한다. ( 구분하기 )
5. replace 함수는 중복되지 않는 범위에서 치환하기 때문에, 자연스럽게 중복이 방지된다.
