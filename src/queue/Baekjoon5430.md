# BOJ_5430_AC

- **알고리즘:** 덱(Deque)   
- -**시간 복잡도** : O(T * N)
- **난이도** : Gold 5

## 핵심
- 뒤집기를 덱의 포인터를 바꿔가면서 처리한다. 
- 포인터의 상태를 isReversed로 처리한다. ( 뒤집혀있냐, 아니냐 )

## 주의
- isRevered 갱신시에는 !isRevered를 쓴다. ( 단순히 true, false 시 실수 가능성 높음 )
- 입력 형식과 출력 형식을 맞출 때는 보편적인 기준과 예외적인 기준을 다 체크한다.

## 피드백
- Deque을 선택한 이유 : 양쪽 끝에서 O(1)으로 제거가 가능한 것이 덱 밖에 없음.
- Queue, Stack을 쓰면 O(1)에 제거 처리는 불가능
