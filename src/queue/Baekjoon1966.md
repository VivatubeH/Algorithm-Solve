# BOJ_1966_프린터 큐

- **알고리즘** : 큐(Queue)
- **시간 복잡도** : O(T * N^2)
- **난이도** : Silver 3 

## 핵심
- O(1)에 삽입, 삭제를 위해 큐를 활용한다.
- 우선순위 뿐만 아니라 인덱스도 써야한다. ( 중요도만으로는 구분 불가 )

## 주의
- 매번 큐를 초기화 해줘야 한다. ( 다음 번에도 큐를 재사용해야 함 )

## 피드백
- 우선순위 큐를 활용해서 우선순위를 별도로 관리하는 방법도 가능하다.
- clear()는 모든 요소를 제거하고 크기를 0으로 만드는 메서드이다. ( 단, O(N)이 추가로 걸림. )
