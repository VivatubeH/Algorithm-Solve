- 유형 : 배열
- 성 지키기

##### 문제 해석
1. 모든 행과 모든 열에 한 명 이상의 경비원 ( 성공 조건 )
2. 최소로 추가해서 성공 조건을 달성할 수 있는 경비원 수 구하기
3. 가로 크기 N, 세로 크기 M이 각각 최대 50.
4. .은 빈칸, X는 경비원이 있는 칸

##### 풀이
1. 행 단위로 경비원이 없는 행이 있는가?
2. 열 단위로 경비원이 없는 열이 있는가?
3. 경비원이 없는 행과, 경비원이 없는 열 중 더 큰수만큼이 경비원이 필요.
- 만약 경비원이 없는 행이 3개, 경비원이 없는 열이 2개라면
- 행과 열이 일치하는 곳에 경비원을 두면 (행, 열)에 대해 처리 가능.
- 이게 최소가 된다.

##### 포인트 
1. 핵심은 경비원이 있는지 없는지지 몇명인지가 아니다.
2. 그래서 행, 열을 체크하는 boolean 배열을 생성한 뒤 false( 경비원 없음)를 센 뒤
3. 더 큰 값만 체크하면 효율적으로 풀 수 있다.
