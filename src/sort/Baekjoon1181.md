# 📂 [Algorithm] 백준 1181 - 단어 정렬

> **관련 링크**: https://www.acmicpc.net/problem/1181
> **난이도**: Silver 5
> **학습 날짜**: 2026-01-19

---

## 1. 문제 해석 🔍
* **목표**: 단어를 길이 짧은 순, 길이 같을 시 사전 순 정렬하기 ( 단, 중복 단어 제거 )
* **입력 값**: 단어 개수 1개 ~ 2만개, 문자열 길이는 최대 50자
* **제약 사항**: 시간 제한 2초, 메모리 제한 256MB
* **핵심 포인트**: 중복 단어 제거하기

## 2. 해결 전략 (아이디어) 💡
* **사용 알고리즘**: 정렬
* **풀이 프로세스**:
    1. Set을 이용해 입력시부터 단어의 중복을 제거한다.
    2. Collections.sort의 정렬 기준으로 String의 메서드를 이용해, 길이 비교 / 사전순 비교를 수행하는 비교
    3. 버퍼에 담고 한번에 출력 ( 시간 복잡도 줄이기 )
* **핵심 아이디어**: String 정렬 메서드 활용

## 3. 복잡도 분석 ⏳
* **시간 복잡도**: O(N logN x L) : 단어 정렬
* **공간 복잡도**: String을 담는 Set, List

## 4. 핵심 코드 리뷰 ⭐
* **중복 제거**: Set을 통해 중복을 제거할 수 있음.

## 5. 트러블슈팅 / 느낀 점 🚩
* **개선 방향**: 람다식을 활용하는 법을 연습하자.
* **배운 점**: 람다식을 알게 되었고, BufferedWriter에 newLine() 메서드를 알게되었다.
