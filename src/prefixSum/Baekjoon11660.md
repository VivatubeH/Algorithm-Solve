# 📂 [Algo] 백준 11660 - 구간 합 구하기 5

> 날짜: 2026-01-23 | 난이도: Silver 1

### 1. 핵심 아이디어 💡
* **알고리즘**: 누적합 중 2차원 누적합
* **핵심 발상**: 
- **전처리 안할 경우** : O(N * N * M) - N이 1024, M이 10만처럼 데이터가 클 시 시간 초과
- **전처리 할 경우** : O(N * N + M) - N이 1024이어도 전처리 이후는 O(1)에 조회 가능

### 2. 코드 리뷰 & 로직 ⭐
* **주의점**
* - 모든 표가 1000으로 채워질 경우, 배열의 원소가 1024 x 1024개가 되어도
* - 10억을 조금 넘는 수준이므로 해당 경우 누적합을 int 배열로도 가능

* **핵심 수식**: 
* - **누적합 전처리** : `S[x, y] = S[x-1, y] + S[x, y-1] - S[x-1, y-1] + V[x, y]`
* - **구간합 조회** : `A[x1, y1][x2, y2] = S[x2, y2] - S[x1 - 1, y2] - S[x2, y1 - 1] + S[x1 - 1, y1 - 1] `

### 3. 트러블슈팅 & 배운 점 🚩
- 누적합 처리 시, 인덱스는 기본 1-based indexing
