# 📂 [Algo] 백준 11659 - 구간 합 구하기 4

> 날짜: 2026-01-22 | 난이도: Silver 3

### 1. 핵심 아이디어 💡
* **알고리즘**: 누적합(PrefixSum) 배열 생성을 통한 시간 복잡도 줄이기
* **핵심 발상**: 
- 매번 더하는 로직 : `O(N * M)` - N과 M이 최대 10만이므로 시간 초과
- 누적합 배열 전처리 후 체크하는 로직 : `O(N)` - 초기 누적합 전처리 / 이후는 O(1)에 검색

### 2. 코드 리뷰 & 로직 ⭐
* **주의점**: i와 j가 같을 수도 있다. 1000 x 100000 = 1억이므로 int로도 가능.
* **핵심 수식**: 
- `S[i, j] = S[j - 1] + A[i]` // 이전까지의 합에서 지금수만 더하면 누적합
- `A[i, j] = S[j] - S[i - 1]` // j까지의 합에서 i-1까지의 합을 빼면 i부터 j의 구간합

### 3. 트러블슈팅 & 배운 점 🚩
- sum 변수를 처음에 두려했는데, 어차피 이전에 구해진 누적합을 다음에 활용하니 불필요.
- 누적합에선 index처리, 합의 범위를 잘 고려해야 할 것 같다.
