# 📂 [Algorithm] 백준 10250 - ACM 호텔

> **관련 링크**: https://www.acmicpc.net/problem/10250
> **난이도**: Bronze 3
> **학습 날짜**: 2026-01-17

---

## 1. 문제 해석 🔍
* **목표**: N번째로 도착한 손님에게 배정될 방 번호
* **입력 값**: 테스트 데이터 T개, H, W, N은 각각 1~99의 자연수
* **제약 사항**: 시간 제한 1초, 메모리 제한 256MB
* **핵심 포인트**: 층과 호수를 12층 3호일때 1203처럼 출력하는 형식

## 2. 해결 전략 (아이디어) 💡
* **사용 알고리즘**: 브루트포스
* **풀이 프로세스**:
    1. 입력받는 층수, 방수, 손님을 기반으로 호실을 반환하는 메서드를 생성한다.
    2. 매 경우 메서드를 통해 호실을 문자열로 반환하고, 버퍼에 기록한다.
    3. 버퍼에 기록한 결과를 마지막에 한 번에 출력한다.
* **핵심 아이디어**: W, N이 100 이하이기 때문에 O(N x M)으로 1만회, 완전 탐색 가능

## 3. 복잡도 분석 ⏳
* **시간 복잡도**: O(N x M) - 층수와 방수만큼 count를 매번 한다.
* **공간 복잡도**: int 변수 몇 개로 효율적. 

## 4. 핵심 코드 리뷰 ⭐
* **findRoomNumber**: 층수, 방수, 손님 번호를 토대로 방 번호를 문자열로 반환함

## 5. 트러블슈팅 / 느낀 점 🚩
* **실수했던 점**: 층수, 호실 수 변수명 선정이 중간에 조금 헷갈렸다. 좀 더 명확해야 함.
* **개선 방향**: 시간 여유가 있으니 브루트포스를 썼지만, 수학적으로도 풀 수 있음.
* **배운 점**: 알고리즘 학습이 아니면 수학적 풀이를 쓰는 게 좋을 것 같다.
