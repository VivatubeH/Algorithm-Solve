# BOJ_10799_쇠막대기

- **알고리즘** : 스택(Stack)
- **시간 복잡도** : O(L)
- **난이도** : Silver 2

## 핵심
- 레이저와 쇠막대기를 어떻게 구분할것인가?
- O(1)에 최근 들어간 레이저를 구분하는 법 -> 스택(Stack)

## 주의
- 만약 유효하지 않은 입력이 있었다면, stack.isEmpty() 체크가 필요했다. 
- 반복된 입력이었다면 stack.clear()가 필요했다.

## 피드백
- 이전 입력을 char 변수 하나로 관리했으면, 메모리를 조금 더 절약할 수 있었다.
- 굳이 스택을 쓸 필요가 없는 경우면 사용하지 않는 게 더 효율적이다.
